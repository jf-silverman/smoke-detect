# smoke_detect

Wildfire smoke detection — data science portfolio project.

## Status

Research phase: surveying public datasets, existing projects, modeling approaches,
evaluation practice, and the operational literature before committing to a design.

## Layout

- `research/` — findings from the research phase (one file per topic)
- `reports/` — synthesized report and recommendations
- `data/` — datasets (gitignored)
- `notebooks/` — exploration
- `src/` — training / eval code
